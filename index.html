dino.sh

Docker Development init for Neos and Flow

View project on GitHub

docker ./dino.sh

Prerequisites

  1. http://brew.sh/

    brew update

brew install wget

  2. https://www.virtualbox.org/

  3. https://www.docker.com/docker-toolbox

  4. Start "Docker Quistart Terminal"

  5. Create a VirtualBox Machine

    docker-machine create --driver virtualbox --virtualbox-disk-size "20000" --virtualbox-memory "4096" --virtualbox-cpu-count "8" default

Add Docker to an existing Project

  1. Download dino.sh to the Project root from: 

    wget https://raw.githubusercontent.com/sbruggmann/dino.sh/master/dino.sh

  2. Fix File permissions:

    chmod +x dino.sh

  3. Add "dino.sh" to Versioning

  4. Add "docker*" to .hgignore

  5. Add the link to this Repository to your Project-Readme:

    https://github.com/sbruggmann/dino.sh

Initialize Docker

./dino.sh

dino.sh Funtion

Show Help

./dino.sh help

Show current Version

./dino.sh version

Reload Docker Project Containers and Images

(Min. required Version is: 201601.7)

./dino.sh reload

To delete all docker files first use:

./dino.sh reload --force

Update to latest init Version

(Min. required Version is: 201601.7)

./dino.sh selfupdate
./dino.sh reload --force

Login to docker container as ..

(Min. required Version is: 201601.13)

.. www-data:

./dino.sh ssh

.. root:

./dino.sh ssh root

Symlinks

(Min. required Version is: 201602.1)

.. show existing symlinks:

./dino.sh links

.. create a symlink:

./dino.sh link www/Neos/Packages/Application

.. remove a symlink:

./dino.sh unlink www/Neos/Packages/Application

Copy Files and Folders

(Min. required Version is: 201602.1)

./dino.sh copy www/Neos/Packages/Application

Run Docker

if Load the Containers:

docker-compose up -d

Connect Container:

docker exec -it PROJECT_main_1 bash

Run Installer (if needed):

/docker/bin/run.sh

Open Site:

http://xyz:8000

Work with Docker

Check other running Containers:

docker ps

Stop running Containers:

docker stop CONTAINER_NAME_OR_ID ANOTHER_CONTAINER_NAME_OR_ID

Delete Docker Containers:

docker rm -f CONTAINER_NAME_OR_ID ANOTHER_CONTAINER_NAME_OR_ID

List Docker Images:

docker images

Delete Docker Images:

docker rmi -f IMAGE_ID ANOTHER_IMAGE_ID
